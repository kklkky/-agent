## 实验1：平行板电容器电场分布分析

### 一、实验目的

1. 掌握恒定电场边界条件及基于高斯定律求解平行板电场的方法
2. 了解介质参数对平行板电容器电容的影响

### 二、预习内容

1. 待求解的平行板结构：电压 U，内导体半径、外导体内外半径分别为 a、b 和 c，内外导体间的介质被平分为上下两部分，磁导率分别为 μ₁、μ₂
2. 预习基于高斯定律求解静电场的方法、静电场的边界条件
3. 熟悉 MATLAB 的基本操作，预习 MATLAB 的画图方法

### 三、实验原理

#### 3.1 忽略介质导电性（静电场情况）

根据边界条件：
- n×(E₁-E₂)=0，电场强度切向连续
- n·(D₁-D₂)=0，电位移法向连续

设 D₁=D₂=D，则有：
- E₁ = D/ε₁
- E₂ = D/ε₂

电压关系：U = E₁·d₁ + E₂·d₂ = D·(d₁/ε₁ + d₂/ε₂)

解得：
```
D = U / (d₁/ε₁ + d₂/ε₂)
```

从而：
```
E₁ = U / (d₁ + (ε₁/ε₂)d₂)
E₂ = U / (d₂ + (ε₂/ε₁)d₁)
```

电容公式：
```
C = Q/U = ε₁ε₂S / (ε₁d₂ + ε₂d₁)
```

#### 3.2 考虑介质导电性（恒定电场情况）

当 σ₁ ≠ 0，σ₂ ≠ 0 时：
- n×(E₁-E₂)=0
- n·(J₁-J₂)=0，电流连续

设 J₁=J₂=J，则：
```
E₁ = J/σ₁
E₂ = J/σ₂
```

电压关系：U = E₁·d₁ + E₂·d₂ = J·(d₁/σ₁ + d₂/σ₂)

电导公式：
```
G = σ₁σ₂S / (σ₁d₂ + σ₂d₁)
```

3. 考虑导电性时，电导与电导率的比值有关，介质电导率越大，电导越大
4. MATLAB 仿真可以直观地展示电磁场分布特性，有助于理解理论公式

## 实验2：同轴线恒定磁场分布分析

### 一、实验目的

1. 掌握恒定磁场边界条件及基于安培环路定律求解恒定磁场的方法
2. 了解介质参数对同轴线恒定磁场分布和外自感的影响

### 二、预习内容

1. 待求解的同轴线结构：内导体通有均匀分布的电流 I，内导体半径、外导体内外半径分别为 a、b 和 c，内外导体间的介质被平分为上下两部分，磁导率分别为 μ₁、μ₂
2. 预习基于安培环路定律求磁场的方法、恒定磁场的边界条件
3. 熟悉 MATLAB 的基本操作，预习 MATLAB 的画图方法

### 三、实验原理

#### 3.1 磁场分布

根据安培环路定律：∮H·dl = I

对于同轴线结构，各区域的磁场强度为：

**区域1（内导体，0 ≤ ρ ≤ a）：**
```
H₁ = Iρ/(2πa²)
B₁ = μ₁H₁
```

**区域2（介质1，a ≤ ρ ≤ b）：**
```
H₂ = I/(2πρ)
B₂ = μ₁H₂
```

**区域3（介质2，b ≤ ρ ≤ c）：**
```
H₃ = I/(2πρ)
B₃ = μ₂H₃
```

**区域4（同轴外部，ρ > c）：**
```
H₄ = 0
B₄ = 0
```

#### 3.2 外自感计算

单位长度外自感：
```
L = (Φ_inner + Φ_outer)/I
  = (μ₁/2π)ln(b/a) + (μ₂/2π)ln(c/b)
```

### 四、实验结果与分析

#### 4.1 基本参数

| 参数 | 数值 |
|------|------|
| 电流 I | 1 A |
| 内导体半径 a | 0.375 mm |
| 外导体内半径 b | 3.625 mm |
| 外导体 外半径 c | 3.75 mm |
| 介质1相对磁导率 μ_r1 | 2 |
| 介质2相对磁导率 μ_r2 | 1 |

#### 4.2 磁场分布结果

**磁场强度随半径变化：**

![图4 磁场强度H随半径ρ的变化](图4_磁场强度变化.png)

**磁感应强度随半径变化：**

![图5 磁感应强度B随半径ρ的变化](图5_磁感应强度变化.png)

**分析：**
1. 在内导体内部（ρ < a），磁场强度 H 随半径 ρ 线性增加
2. 在介质区域（a < ρ < c），磁场强度 H 与 1/ρ 成反比
3. 在同轴外部（ρ > c），磁场强度 H 降为 0
4. 磁感应强度 B 在不同介质区域有明显突变，这与磁导率的不同有关

#### 4.3 外自感分析

当 μ_r1 = μ_r2 = 1 时：
- 内导体区域自感贡献: 1.726e-7 H/m
- 外导体区域自感贡献: 1.727e-7 H/m
- 单位长度总外自感: 3.453e-7 H/m = 0.3453 μH/m

**外自感随磁导率变化：**

![图6 外自感随磁导率μ_r1的变化](图6_外自感随磁导率变化.png)

**分析：**
1. 当 μ_r1 从 1 增大到 2 时，外自感 L 线性增加
2. 磁导率越大，磁场能量密度越大，因此自感越大
3. 当 μ_r1 趋近于 μ_r2 时，内外介质区域的磁导率趋于一致

### 五、实验结论

1. 同轴线的磁场分布遵循安培环路定律，在内导体内部线性分布，在外部与 1/ρ 成反比
2. 磁感应强度在不同介质交界处发生突变，与边界条件一致
3. 同轴线的外自感与磁导率成正比，磁导率越大，自感越大
4. 当两种介质的磁导率趋于一致时，磁场分布更加均匀

## 附录：主要程序

### 实验1程序 (experiment1.m)

```matlab
%% 实验1：平行板电容器电场分布分析
clear; clc; close all;

%% 基础参数设置
U = 1;          % 电压 (V)
S = 100;        % 面积 (mm^2)
d1 = 2;         % 介质1厚度 (mm)
d2 = 4;         % 介质2厚度 (mm)

% 将单位转换为米
S_m = S * 1e-6;       % mm^2 -> m^2
d1_m = d1 * 1e-3;      % mm -> m
d2_m = d2 * 1e-3;      % mm -> m

%% 第一部分：忽略介质导电性 (静电场)
eps0 = 8.854e-12;  % F/m

% 固定 er1 = 1, er2 从 1 变化到 10
er1_fixed = 1;
er2_range = 1:0.1:10;
C_part1 = zeros(size(er2_range));

for i = 1:length(er2_range)
    er2 = er2_range(i);
    eps1 = er1_fixed * eps0;
    eps2 = er2 * eps0;
    C = (eps1 * eps2 * S_m) / (eps1 * d2_m + eps2 * d1_m);
    C_part1(i) = C * 1e12;  % 转换为 pF
end

% ... (完整代码见 experiment1.m)
```
